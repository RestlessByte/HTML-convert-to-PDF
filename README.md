# Convert URL Website to PDS
## How using?:
**Example**
```bash
git clone git@github.com:RestlessByte/HTML-convert-to-PDF.git
cd HTML-convert-to-PDF 
&& bun install 
&& bun index.ts https://example.com
```
***You get file in root dir with name *output.pdf***
